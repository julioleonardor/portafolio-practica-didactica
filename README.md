# Portafolio de Práctica Didáctica

Sitio estático preparado para GitHub Pages.

## Publicación
1. Cree un repositorio en GitHub.
2. Suba todo el contenido de esta carpeta a la rama `main`.
3. Entre a **Settings → Pages**.
4. Seleccione **Deploy from a branch**.
5. Elija `main` y `/root`.

## Pendiente antes de entregar
Edite `index.html` y sustituya los seis textos "Escribe aquí tu respuesta final" por las respuestas de autorreflexión.
